Modulo di correzione file


La seguente pagina permette di elaborare il file IE815 (Draft DAA Telematico) perchè venga accettato dal Servizio telematico doganale. E' possibile effettuare il caricamento attraverso il pulsante 'Browse' nell'area Upload file. Il contenuto del file appena caricato è visualizzabile nell'area sottostante. Cliccando il pulsante Send verrà scaricato il file con le correzioni necessarie. Tale file avrà il nome e l'estensione del file originale con l'aggiunta della stringa '_ELAB', che deve essere rimossa dall'utente. Una volta rinominato il file è necessario firmarlo, tramite il programma di firma in uso, e inviarlo al Servizio telematico doganale per il rilascio dell'ARC.

N.B. Nel caso in cui non sia visibile la stringa '_ELAB', che deve essere rimossa, è necessario impostare correttamente le proprietà del PC in uso perchè siano visibili le estensioni dei file.
